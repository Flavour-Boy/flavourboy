# LCD Wiring

## LCD Pins

- Pin 1 → GND
- Pin 2 → 5V
- Pin 3 → 3k resistor → GND
- Pin 4 → Nucleo D1
- Pin 5 → GND
- Pin 6 → Nucleo D2
- Pin 7 → Not connected
- Pin 8 → Not connected
- Pin 9 → Not connected
- Pin 10 → Not connected
- Pin 11 → Nucleo D3
- Pin 12 → Nucleo D4
- Pin 13 → Nucleo D5
- Pin 14 → Nucleo D6
- Pin 15 → 100Ω resistor → 5V
- Pin 16 → GND

## Buttons

- Left button → Nucleo D8
- Right button → Nucleo D9
