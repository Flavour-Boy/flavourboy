# LCD Wiring

## LCD Pins

1 → GND
2 → 5V
3 → Potentiometer middle pin
    • Left pin → GND
    • Right pin → 5V
4 → Nucleo D1
5 → GND
6 → Nucleo D2
7 → Not connected
8 → Not connected
9 → Not connected
10 → Not connected
11 → Nucleo D3
12 → Nucleo D4
13 → Nucleo D5
14 → Nucleo D6
15 → 100Ω resistor → 5V
16 → GND

---

## Buttons

Left button → Nucleo D8
Right button → Nucleo D9
